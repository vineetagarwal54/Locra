# Locra Screen Reference Map

## Onboarding

### Welcome
Reference: `design/references/onboarding-welcome.png`

Behavior:
- First launch only.
- Continue opens Benefits.
- Ambient background animation defined in MOTION.md.

### Benefits
Reference: `design/references/onboarding-benefits.png`

Behavior:
- Explains offline, privacy, and local responses.
- Continue opens Model Introduction.

### Model Introduction
Reference: `design/references/model-intro.png`

Behavior:
- Shows actual model metadata.
- Download Model begins download flow.
- Notification explanation is shown before requesting Android notification permission.

### Notification Explanation
Reference: `design/references/notification-explanation.png`

Behavior:
- Appears contextually after Download Model.
- Allow Notifications triggers Android system permission request.
- Not Now continues download without notification permission.

## Chat

### New Chat
Reference: `design/references/new-chat.png`

Behavior:
- Text input available immediately.
- Suggestions insert or submit prompts.
- Attachment opens camera/gallery actions.
- Drawer icon opens conversation drawer.

### Active Chat
Reference: `design/references/active-chat.png`

Behavior:
- Conversation scrolls behind fixed composer.
- No bottom tab bar.
- Last message must scroll completely above composer.

### Image Preview
Reference: `design/references/image-preview.png`

Behavior:
- Attached image shown before send.
- Image removable.
- Same composer used for text and image questions.

### Image Answer
Reference: `design/references/image-answer.png`

Behavior:
- Image question and assistant answer remain in one conversation.
- Normal text follow-ups continue from same composer.

## Navigation

### Conversation Drawer
Reference: `design/references/conversation-drawer.png`

Behavior:
- New Chat.
- Recent conversation switching.
- View All History.
- Settings.

### History
Reference: `design/references/history.png`

Behavior:
- Full screen.
- Grouped by Today, Yesterday, Previous 7 Days.
- One item per conversation.
